Dr. Mark Tee

No pre-requisite needed / Bring Laptop

30 November 2019

2:00 PM - 5:00 PM

A303 IoT Lab

The heart of every Internet-enabled solution is a Microcontroller, and this workshop will acquaint you with the hardware, setting up Raspbian Operating System, basic command line file system navigation and how to remotely access the RPi through the network in "headless" mode.

Title: Git Gud with Github - Introduction to version control

Fu Swee Tee

No pre-requisite needed / Bring Laptop

7th Dec 2019 (Saturday)

9:00 AM - 12:00 PM

B219

In this workshop, you will be introduced to the basics of version control, understand why it is useful and implement basic version control for project solutions using “GitHub Desktop”, a Graphical User interface for GitHub. You will be able to explore GitHub desktop in simplifying your development workflow, through a visual interface instead of text commands on the command line.

Register: https://bit.ly/2Q1koCq
